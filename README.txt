Simple personal project of mine, practicing the use of pandas and regex

This Python script scans for email addresses or phone numbers in your text, made using Python 3.9, using pandas and re

1. Dump text to be matched in 'data.txt'
2. Run script
3. Enter 'E' to match email addresses or 'N' to match phone numbers(Singapore)
4. Open the new excel file created in the same directory.
